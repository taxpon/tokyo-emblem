### addtional info
You can tweak the emblem height, base layer heigth and size of emblem.

| Item                   | Variable name in code |
|------------------------|-----------------------|
| Emblem height (mm)     | em_height             |
| Base layer height (mm) | base_height           |
| Size of emblem (mm)    | unit                  |
